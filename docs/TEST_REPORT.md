  - Level3 Boss：隐身、反弹、召唤高级敌人
  - Final Boss：激光、炸弹、机枪、召唤、反弹、空袭、护盾
- 已补充

  - 刷新点重新向四角分布，地图开场节奏更直观
- 已通过内建自检验证
  - Level2 Boss 会生成炸弹预警并触发机枪扫射
  - Level3 Boss 会进入隐身或反弹状态
  - Final Boss 会触发空袭预警和激光特效
  - LaserGuard 会刷新护盾并释放激光
  - BombThrowEnemy 会生成四向投弹预警
  - Kamikaze 会先蓄力预警，再在接近 2 秒后自爆
